Software

Here you can find all software implementations that I have been involved. Implementations feature Wireless Sensor and Actuator Networks (WSANs), the HVAC testbed and the Smart Mobility Lab testbed.

  • GISOO - A co-simulator integrating Simulink and COOJA, allowing the implementation of Wireless Networked Control Systems.

  • KTH-WSN - Source code and documentation for Wireless Sensor and Actuator Networks. IEEE 802.15.4 MAC and Routing protocols, several drivers for Telosb, Tmote Sky and CM5000 motes, Arduino-Mote interface, Getting started with TinyOS, etc.

  • KTH-HVAC - Source code and documentation for the KTH HVAC testbed

  • KTH-SML - Source code and documentation for the KTH Smart Mobility Lab
