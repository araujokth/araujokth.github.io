Software

Here you can find all software implementations that I have been involved. Implementations feature Wireless Sensor and Actuator Networks (WSANs), the HVAC testbed and the Smart Mobility Lab testbed. Since I moved to Ericsson Research, I have not maintained this software.

  • GISOO - A co-simulator integrating Simulink and COOJA, allowing the implementation of Wireless Networked Control Systems. As of April 2017, the GISOO simulator is working and being used by A. Luppi from TU Graz.

  • KTH-WSN - Source code and documentation for Wireless Sensor and Actuator Networks. IEEE 802.15.4 MAC and Routing protocols, several drivers for Telosb, Tmote Sky and CM5000 motes, Arduino-Mote interface, Getting started with TinyOS, etc. (Not active)

  • KTH-HVAC - Source code and documentation for the KTH HVAC testbed (Not active)

  • KTH-SML - Source code and documentation for the KTH Smart Mobility Lab (Not active)
